# コマンドパレットからコミットする方法

## 🎯 テキストボックスが表示されない場合の対処法

### 方法1: コマンドパレットからコミット（確実）

1. **コマンドパレットを開く**
   - `Ctrl + Shift + P` を押す

2. **「commit」と入力**
   - 「**Git: Commit**」または「**Git: Commit (Staged)**」を選択

3. **コミットメッセージを入力**
   - ポップアップウィンドウまたはテキストボックスが表示されます
   - メッセージを入力（例：`Initial commit: Sappy Study Abroad Platform`）
   - Enterキーを押す

これでコミットが完了します！

---

### 方法2: ステージング状況を確認

もしかすると、ファイルがステージングされていない可能性があります。

1. **ソース管理パネル（📊）を開く**
2. **「Staged Changes」セクションを確認**
   - ファイルが表示されているか？
   - 空の場合は、まだステージングされていません

3. **再度ステージング**
   - コマンドパレット（`Ctrl + Shift + P`）→ 「**Git: Stage All Changes**」

---

### 方法3: 個別にファイルを確認

1. **ソース管理パネルで「Changes」セクションを確認**
2. **各ファイルの横に「+」ボタンがあるか確認**
3. **「+」ボタンをクリックして個別にステージング**

---

## ✅ 推奨手順

**今すぐ試してください：**

1. `Ctrl + Shift + P` を押す
2. 「**commit**」と入力
3. 「**Git: Commit**」を選択
4. メッセージを入力してEnter

これでコミットできるはずです！
