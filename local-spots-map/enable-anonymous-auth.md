# 匿名認証を有効にする方法

Supabaseで匿名認証を有効にする必要があります。

## 手順

1. Supabaseダッシュボードにログイン
2. プロジェクトを選択
3. 左側のメニューから **Authentication** をクリック
4. **Providers** タブをクリック
5. **Anonymous** を探して、**Enable** をクリック
6. 設定を保存

## 確認方法

ブラウザのコンソールを開いて、コミュニティに参加しようとすると、以下のようなログが表示されます：

- ✅ 匿名認証成功: [ユーザーID]
- ❌ 匿名認証エラー: [エラー詳細]

エラーが表示される場合は、上記の手順を確認してください。
