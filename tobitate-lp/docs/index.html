トビタテ18期秋田対策コミュニティ 使い方説明書

まず最初に、一番大切なこと：

  • ここは参加者側が自由にサポートを求める場です
  • 分からないことはいつでも聞いてください
  • サポートが欲しいときはいつでも求めてください
  • 質問の完成度・レベル・進捗は一切問いません
  • 「こんなこと聞いていいのかな？」と思う必要はありません

1. このコミュニティについて

このコミュニティは、トビタテ18期の応募を目指す方（参加者）と、過去期のトビタテ生（サポーター）が集まり、応募準備をサポートし合う場です。上下関係のない、フラットな支援コミュニティとして運営されています。

構成メンバー

  • 参加者：トビタテ18期の応募を目指している方
  • サポーター：過去期のトビタテ生（名前の後ろに「◯期」と表示）

2. 使い方

基本姿勢

  • 困ったら、迷ったら、まずコミュニティで聞く
  • 過去ログを探すのも良いが、探しすぎる必要はない
  • 一人で考え続けず、「頼る前提」で使っていい場

Slackの使い方

  • メインチャンネル：「全体＿情報共有」
    • 重要な情報・共有・質問は基本ここに集まります
  • 過去ログ・ピン留め・フォルダは確認してもよいが、見つからなくても質問してOK

サポートの受け方

  • 全体チャンネルで質問：他の参加者も同じ疑問を持っている可能性がある
  • DMで個別相談：「みんなの前で聞くのは...」と思う場合は、サポーターに直接DM
  • 素直に書く：「壁打ちしてほしい」「計画を見てほしい」と遠慮なく書いてOK

相談できること（例）

  • 留学計画のテーマが固まらない段階での相談
  • トビタテの募集要項・制度についての質問
  • 過去の通過事例や計画の方向性についての共有依頼
  • 留学計画書の構成や考え方についての壁打ち
  • 計画書を見てほしいという依頼
  • 面接対策（想定質問、答え方の考え方）
  • モチベーションが下がっているときの雑相談
  • 「何から始めればいいかわからない」という初期段階の相談

3. サポーターとの関わり方

サポーターの見つけ方

  • 名前の後ろに「◯期」と表示されているのがサポーター
  • サポーター情報は、全体＿情報共有チャンネルのフォルダにまとまっている

サポーターへの連絡方法

  • DMで直接連絡：「この方の留学計画に近いかも」と思ったら、遠慮なくDM
  • 全体チャンネルで聞く：「◯◯のような計画を考えているのですが、経験のあるサポーターの方いらっしゃいますか？」

サポーター側のスタンス

  • こちらから無理に介入することはしない
  • 情報共有をベースに、必要に応じて個別対応する

4. 参加者同士の相互サポート

このコミュニティでは、参加者同士のやり取り・助け合いも歓迎しています。

  • 同じ立場だからこそできる情報共有や共感、気づきがある
  • 他の参加者の質問に「自分なりの考え」や「調べたこと」を返してもよい
  • 「まだ応募者だけど、ここは分かる」という立場での発言もOK
  • 完璧な答えでなくても、あなたの視点や経験は価値がある

5. 最後に

この説明書は「ガイド」であり、「ルール」ではありません。このコミュニティには厳格なルールはなく、あなたが使いやすい形で使ってください。

迷ったら、まず聞いてください。

このコミュニティは、あなたが一人で抱えなくていい場所です。遠慮せず、どんどん頼ってください。
